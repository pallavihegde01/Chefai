# ChefAI — AI-Powered Recipe Generator

**Live Demo:** [ChefAi](https://chefai-7rwy.onrender.com)

ChefAI is a full-stack web application that helps users generate creative recipes based on the ingredients they have at home.  
Simply type in your ingredients, click **"Generate Recipe"**, and get an instant AI-suggested recipe — all in a clean, responsive interface.

---

## Features

- **AI Integration** — Generates unique recipes using your listed ingredients.  
- **Ingredient Management** — Add, remove, or clear ingredients before generating.  
- **Smooth User Experience** — Animated scrolling and responsive layout.  
- **Full Stack Setup** — React frontend + Express backend.  
- **Modern UI** — Built with Tailwind CSS and React Icons.

---
